hexo部署小记

参照官方文档重新部署了hexo.将2014年的几篇记录迁移过来了，准备再开始记录一些学习笔记。

安装提示

  • hexo init 一定要在空目录执行
  • 安装完成后，执行 npm install 安装依赖的插件（如果不执行这一步，功能看似正常，但是无法渲染markdown）
  • 由于网络原因，建议将 npm 改为 cnpm,cnpm安装指南
