背景

电脑中收藏的开源项目太多，svn、git托管的都用，在系统目录中保存，近期电脑硬盘不够用，打算将这些项目的源代码删除，只保留开源地址记录。

思路

依次遍历目录下的子目录，判断其是否是用版本管理软件管理的代码文件夹，如果是则将该项目名称及开源URL保存到日志文件中。

获取git项目的URL

通过查看git help文档可知，在仓库目录下执行‘git remote -v’可以获取当前仓库的远程地址。
